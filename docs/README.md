# Documentation

Lune documentation is organized by purpose:

- `docs/specs/`: product and architecture specifications
- `docs/guides/`: operational and contributor guides
- `docs/plans/`: implementation plans and work breakdowns

## Start Here

- Spec: `docs/specs/sdd.md`
- Theming guide: `docs/theming.md`
- Testing guide: `docs/guides/testing.md`
- Implementation plan index: `docs/plans/sdd/00-index.md`
