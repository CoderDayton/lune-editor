# Project Guide

## Rust Optimizations

Rust 2025-2026 optimization compact digest: Profile-guided optimization (PGO) for hot binaries yields double-digit speedups; use faster linkers (mold/lld/Sold) and Cranelift for compile speed; parallelize with Rayon scoped threads leveraging available_parallelism; tune Cargo.toml profiles (LTO, codegen-units=1, opt-level=3, panic=abort); design cache-friendly data structures with stack arrays and preallocated Vecs to minimize heap traffic; profile with perf/flamegraphs and optimize only proven hot paths; run Miri and sanitizers (ASan/TSan) in CI to eliminate UB that blocks optimizer assumptions; reuse buffers and stream data to avoid allocation churn; strip debug-only logging from release builds; monitor rustc-perf metrics and trim proc-macro/dependency bloat for workspace responsiveness. Code/math/data optimizations: Exploit portable_simd with aligned chunk processing and runtime CPU feature detection for vectorized numeric ops (floats require nightly -C target-cpu=native flags to permit reordering); leverage const generics to monomorphize branches at compile-time eliminating runtime checks; inline hot functions with #[inline]/#[inline(always)] to enable cross-function optimization and loop unrolling; use iterators and closures for zero-cost abstractions via static dispatch instead of trait objects; apply asm! inline assembly for critical cryptographic or bit-manipulation kernels where LLVM codegen lags; structure data with #[repr(C)]/#[repr(align)] to control layout and eliminate padding in hot structs; batch computations using SIMD-width chunks (e.g., process 8× f32 per loop with Simd<f32, 8>) and accumulate results at the end; replace bounds-checked indexing with unsafe get_unchecked in verified hot loops after profiling confirms the overhead; use integer math and fixed-point arithmetic where floating-point precision isn't required to avoid FPU stalls; prefer SmallVec, ArrayVec, and stack-allocated fixed-size buffers over heap Vec for collections under 32 elements to eliminate allocator calls.
